# Blinkit Sales Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit's sales performance using interactive visualizations and KPIs.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## KPIs
- Total Revenue
- Total Orders
- Average Rating
- Average Profit
- Total Reviews

## Dashboard Features
- Interactive Slicers
- KPI Cards
- Bar Charts
- Line Charts
- Map Visualization
- Clear Filter Button
